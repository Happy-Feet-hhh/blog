## 1.什么是 WebGPU？

WebGPU 是新一代 Web 图形 API，由 W3C 制定标准，旨在取代 WebGL。它基于现代 GPU API（Vulkan、Metal、Direct3D 12）设计，提供更好的性能、更灵活的 GPU 加速能力，并原生支持通用 GPU 计算（GPGPU）。

在2026年，WebGPU在主流的浏览器上都已经支持，在[https://caniuse.com/webgpu](https://caniuse.com/webgpu)中可以看到，无论是pc还是移动端，WebGPU都已被支持：
![](Pasted%20image%2020260412192325.png)

并且WebGPU除了能在Web上运行，还可以在本地运行，通过[Dawn](https://dawn.googlesource.com/dawn)和[WGPU](https://wgpu.rs/)。由于本人就拥有Rust的背景，所以打算以后通过WGPU去玩一玩本地的webgpu，同时也算为Bevy打基础了。

本项目的初衷是通过学习WebGPU来学习图形渲染的GPU编程，最后以一个游戏项目结束WebGPU的学习。

学习前提：
- 具有一定的代码基础
- 具有nodejs环境

## 2. 环境搭建 —— Vite
本次教程采用 Vite 构建系统，Vite作为主流的构建系统，市场占有度很高，并且在Vite 8版本全面"锈"化，性能也非常不错。

用 Vite 搭建 WebGPU 开发环境只需**5 个核心步骤**，包含项目初始化、类型支持、配置调整、基础示例与运行调试，全程约**5 分钟**即可完成。

### 一、快速创建项目

1. **创建 Vite 项目**
```bash
# 使用npm 
npm create vite@latest webgpu-project -- --template vanilla-ts 
# 或使用pnpm（推荐） 
pnpm create vite webgpu-project -- --template vanilla-ts
```
2. 进入项目并安装依赖
```bash
cd webgpu-project 
npm install # 或 pnpm install
```
3. 安装 WebGPU 类型声明与辅助库
```bash
# WebGPU类型声明，提供完整类型提示 
npm install -D @webgpu/types 
# 可选：矩阵运算库，简化3D数学计算 
npm install wgpu-matrix
```
`@webgpu/types`是开发 WebGPU 必备的类型支持包。

### 二、TypeScript 配置调整
修改`tsconfig.json`，添加 WebGPU 类型支持：
```json
{

  "compilerOptions": {
    "target": "es2023",
    "module": "esnext",
    "lib": ["ES2023", "DOM", "DOM.Iterable", "WebWorker"],  // 确保包含DOM和WebWorker
    "types": ["vite/client", "@webgpu/types"],  // 添加WebGPU类型
    "skipLibCheck": true,

    /* Bundler mode */
    "moduleResolution": "bundler",
    "allowImportingTsExtensions": true,
    "verbatimModuleSyntax": true,
    "moduleDetection": "force",
    "noEmit": true,

    /* Linting */
    "noUnusedLocals": true,
    "noUnusedParameters": true,
    "erasableSyntaxOnly": true,
    "noFallthroughCasesInSwitch": true
  },

  "include": ["src"]
}
```
### 四、编写基础 WebGPU 示例

1. **清理默认文件**
    
    删除`src/counter.ts`和`src/style.css`，简化项目结构。
    ![](Pasted%20image%2020260412193036.png)
    
2. **修改 WebGPU 入口文件（src/main.ts）**
```typescript
async function initWebGPU() {
  // 1. 检查浏览器WebGPU支持
  if (!navigator.gpu) {
    throw new Error('当前浏览器不支持WebGPU');
  }

  // 2. 请求GPU适配器
  const adapter = await navigator.gpu.requestAdapter();
  if (!adapter) {
    throw new Error('无法获取GPU适配器');
  }
  
  // 3. 创建GPU设备
  const device = await adapter.requestDevice();

  // 4. 配置canvas上下文
  const canvas = document.querySelector('canvas')!;
  const context = canvas.getContext('webgpu')!;
  const format = navigator.gpu.getPreferredCanvasFormat();
  context.configure({
    device,
    format,
    alphaMode: 'opaque'
  });

  console.log('WebGPU初始化成功！', device);
  return { device, context, format };
}

// 页面加载完成后初始化
window.addEventListener('load', () => {
  initWebGPU().catch(err => console.error('WebGPU初始化失败:', err));
});
```
以上的代码是一切WebGPU的初始化代码，具体的细节不用在意，主要是通过一些创建操作来看看当前浏览器是否支持WebGPU。

# 一级标题

段落之间用空行分隔。

第二段。_斜体_，**粗体**，和 `等宽字体`。无序列表
如下所示：

- 这个
- 那个
- 另一个

注意——不考虑星号——实际文本内容从第 4 列开始。

> 引用块是
> 这样写的。
>
> 它们可以跨越多个段落，
> 如果你愿意的话。

使用 3 个短横线表示破折号。使用 2 个短横线表示范围（例如，"详见第 12--14 章"）。三个点 ... 将被转换为省略号。
支持 Unicode。☺

## 二级标题

这里是一个有序列表：

1. 第一项
2. 第二项
3. 第三项

再次注意实际文本如何从第 4 列开始（距左侧 4 个字符）。下面是一个代码示例：

    # 让我再重申一下...
    for i in 1 .. 10 { do-something(i) }

你大概猜到了，缩进 4 个空格。顺便说一下，除了缩进代码块，你也可以使用分隔块：

```
define foobar() {
    print "Welcome to flavor country!";
}
```

（这使得复制和粘贴更容易）。你可以选择为分隔块标记语言以启用语法高亮：

```python
import time
# 快，数到十！
for i in range(10):
    # （但不要*太*快）
    time.sleep(0.5)
    print i
```

    - 扁豆

2. 烧一些水。

3. 把所有东西倒进锅里，然后按照
    以下步骤操作：

        找到木勺
        打开锅盖
        搅拌
        盖上锅盖
        把木勺摇摇晃晃地放在锅把上
        等待 10 分钟
        回到第一步（或完成后关火）
